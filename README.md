# TEST SERVER OPEN METHOD
1. cd <FrontEnd Directory>

2. npm install
    `sudo npm install`

3. set .env file
    ```vi .env```
    
    ```REACT_APP_GOOGLE_AUTH_CLIENT_ID="Your_Google_Auth_Clinet_ID"```
    ```REACT_APP_API_URL = "<http://localhost:8000>"```
    ```:wq```    
4. turn on front server
    ```sudo npm start```
